# ペルソナ定義

## Persona 1: 主婦・主夫ユーザー「田中 さおり」

| 項目 | 内容 |
|---|---|
| **名前** | 田中 さおり（35歳） |
| **役割** | 専業主婦（2児の母） |
| **家族構成** | 夫・子供2人（小学生・幼稚園児） |
| **料理スキル** | 中級（レパートリーは20〜30品程度） |

### 背景・状況
毎日3食の献立を考えることに疲弊している。特に夕食は「また同じものか」と家族に言われるのが嫌で、毎日プレッシャーを感じている。料理自体は嫌いではないが、「何を作るか考える」ことが最大のストレス。スマートフォンはよく使うが、複雑なアプリは続かない。

### ペインポイント
- 毎日の献立を考えるのが面倒・ストレス
- 知らないレシピを試して失敗するのが怖い
- 「自分が作れる料理」の中から選びたいのに、既存アプリは知らないレシピばかり提案してくる
- 献立アプリを開いても結局自分で考えてしまう

### モチベーション
- 考えなくていいなら最高
- 自分が作れる料理の中から提案してほしい
- 少し笑えたり、共感してもらえると続けられる
- 「ダメでいい」と言ってもらえると気が楽になる

### 利用シーン
- 朝食後、今日の夕食を考えるとき
- スーパーに行く前に献立を決めたいとき
- 「今日は疲れた、楽したい」と思ったとき

### DAMESIとの関係
- **料理登録**: 作った料理の写真を撮ってアップロード。AIが自動入力してくれるのが嬉しい
- **献立提案**: ボタン1つで決まるのが最高。3品から選べるのでちょうどいい
- **ガチャ**: 「どうせ決められないなら運に任せよう」という気持ちで使う。SSRが出ると少し嬉しい
- **キャラクター**: サボ母ちゃんの全肯定に癒される。たまにシェフレイの叱責で笑える
