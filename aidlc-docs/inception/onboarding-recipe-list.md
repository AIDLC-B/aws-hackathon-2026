# オンボーディング初期料理リスト

初回オンボーディング時に提案する固定料理リスト。
ユーザーが「作れる」と思った料理を選択してレパートリーに追加する。

## 設計方針
- **レアリティ**: 料理ごとにデフォルト値を設定
- **難易度**: 料理ごとにデフォルト値を設定
- **所要時間**: 料理ごとにデフォルト値を設定（後でレシピタブから編集可能）
- **バナナは必須**: スキップしても必ずレパートリーに追加される
- **スキップ**: 可能（0件でも先に進める。ただしバナナは強制追加）
